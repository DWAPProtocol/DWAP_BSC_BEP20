# قائمة فحص نشر DWAP Token على BSC

## قبل البدء

### الإعداد الأساسي
- [ ] تثبيت Node.js (v16 أو أحدث)
- [ ] تثبيت npm أو yarn
- [ ] حساب BSC مع أموال كافية (للغاز)
- [ ] مفتاح private key آمن
- [ ] بيانات اعتماد BscScan API (للتحقق)

### الملفات والمجلدات
- [ ] وجود folder `/contracts` مع جميع الملفات .sol
- [ ] وجود folder `/scripts` مع deploy.js
- [ ] وجود `package.json` مع الاعتماديات
- [ ] وجود `hardhat.config.js` الصحيح
- [ ] وجود `.env` مع القيم الصحيحة
