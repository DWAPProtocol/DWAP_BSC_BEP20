## نقل الملكية للـ DAO (المرحلة 5)

### الخطوة 1: تحضير المقترح
```javascript
// 1. تحضير بيانات المقترح
const targets = [dwapTokenAddress];
const values = [0];
const calldatas = [
  dwapToken.interface.encodeFunctionData('transferOwnership', [governorAddress])
];
const description = "Transfer DWAP Token ownership to DAO Governor";

// 2. إنشاء hash الوصف
const descriptionHash = ethers.id(description);
```

### الخطوة 2: إنشاء المقترح
```javascript
// يجب أن يكون المالك لديه 1+ DWAP ويجب أن يفوض نفسه
const tx = await governor.propose(targets, values, calldatas, description);
```

### الخطوة 3: التصويت
- يستمر التصويت لمدة 7 أيام
- يحتاج على 4% نصاب
- يحتاج على 50%+ موافقة

### الخطوة 4: الانتظار
- بعد انتهاء التصويت، ينتظر Timelock لمدة 2 يوم
- يمكن لأي شخص تنفيذ المقترح بعد انتهاء المدة

### الخطوة 5: التنفيذ
```javascript
const tx = await governor.execute(targets, values, calldatas, descriptionHash);
```
